## Description

Implement JWT authentication: POST /api/v1/auth/signup (create user with hashed password), POST /api/v1/auth/login (return JWT access token), GET /api/v1/auth/me (current user info). Use python-jose for JWT, bcrypt for password hashing. Create auth dependency `get_current_user` for protected routes. User roles: admin, sales, engineer. Write tests for all endpoints.
